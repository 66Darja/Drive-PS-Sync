Drive PS Sync

Drive PS Sync is a personal PowerShell tool created by the developer to synchronize local folders and files with Google Drive.

Purpose

The application is used exclusively by the developer for personal file management and backup automation.

Data Access

The application accesses the user's own Google Drive account to upload, update, or synchronize files. No data is shared with any third parties.

Developer

This application is privately developed and maintained for personal use only.
